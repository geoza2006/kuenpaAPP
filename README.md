# 📱 Kuen Pa (คืนป่า) - Wildlife Rescue App

> **"เปลี่ยนคุณให้เป็นฮีโร่ ช่วยสัตว์ป่าพลัดหลงในคลิกเดียว"**

แอปพลิเคชัน **Kuen Pa (คืนป่า)** พัฒนาขึ้นเพื่อแก้ปัญหาเมื่อประชาชนพบเจอสัตว์ป่าบาดเจ็บหรือพลัดหลงในเขตเมือง โดยเน้นการเชื่อมต่อระหว่างพลเมืองและเจ้าหน้าที่อย่างรวดเร็วผ่านระบบอัจฉริยะ เพื่อสนับสนุนเป้าหมายการพัฒนาที่ยั่งยืน **SDG 15: Life on Land**

-----

## ✨ ฟีเจอร์เด่น (Key Features)

  * 📸 **Smart Reporting:** ถ่ายรูปสัตว์ป่าพร้อมระบบวิเคราะห์อาการเบื้องต้น
  * 📍 **Auto-GPS Tracking:** ดึงพิกัดตำแหน่งที่เกิดเหตุอัตโนมัติ แม่นยำ ไม่ต้องเสียเวลาอธิบายเส้นทาง
  * 🎮 **Education Mini-Games:** เกมให้ความรู้เกี่ยวกับสัตว์ป่าไทย เล่นฆ่าเวลาระหว่างรอเจ้าหน้าที่
-----
